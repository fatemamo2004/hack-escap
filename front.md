# Hack & Escape — Frontend Website (Full Implementation Spec)

## Overview
A modern bilingual (English / Arabic) cybersecurity competition website.

The website represents:
- Hack & Escape competition
- Jordan Cyber Club initiative
- A professional portfolio-grade cyber event platform

## 🌐 Core Features

### 1. Bilingual Support (EN / AR)
- Language toggle in navbar
- Arabic supports RTL layout
- Dynamic text switching
- Font switching:
  - English: Inter / JetBrains Mono
  - Arabic: Cairo / IBM Plex Arabic

---

### 2. Design System

#### Theme
- Dark cyber theme
- Hacker / terminal aesthetic
- Futuristic UI

#### Colors
- Background: #0a0f1a
- Primary: Neon Cyan (#00f5d4)
- Secondary: Electric Blue (#0ff)
- Text: Light (#e6f1ff)

### 3. UI Elements

- Terminal-style text blocks
